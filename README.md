# Task Tracker App
A simple task tracking application built with React. Users can add, delete, and toggle reminders for tasks.

## Features
Add tasks
Delete tasks
Toggle reminders
Responsive design

## Technologies Used
- React
- JavaScript
- CSS
- JSON server (optional)
