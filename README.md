# Task Tracker App
A simple task tracking application built with React. Users can add, delete, and toggle reminders for tasks.

## Features
- Add tasks
- Delete tasks
- Toggle reminders
- Responsive design
- Mark individual tasks complete/uncomplete
- Mark all tasks complete
- Completed state persists to backend
- Task details page
- Visual completed states (line-through + checkbox)

- React
